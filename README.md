# The Simple ToDo App

This is my project for Sprint 7. It is a simple to-do application.

## Functionality

With this app, users can add and delete tasks, assign due dates for each task, and check a box once the task has been completed.

## Technology

- HTML
- CSS
- JavaScript
- OOP
